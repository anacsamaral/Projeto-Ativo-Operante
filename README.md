# Projeto-Ativo-Operante
Sistema Web capaz de oferecer uma ferramenta ao cidadão que deseja avisar ou denunciar aos orgãos competentes sobre problemas na cidade, como buracos na rua, abandono de animais, poda ou corte de árvores, problemas no trânsito, enfim qualquer situação  que pode afetar o dia a dia do cidadão.
